This repository contains a minimal Next.js store for the "Just Bead It" handmade jewelry brand.

## Setup
1. Copy files into a Next.js project and ensure TypeScript is installed.
2. Install dependencies:

```bash
npm install
```

3. Add images to `/public/images/` and run dev server:

```bash
npm run dev
```

4. Add env vars from .env.example and deploy to Vercel.
